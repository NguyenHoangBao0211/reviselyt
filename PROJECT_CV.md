## PROJECTS

**Reviselyt: AI Document Summarizer & Study Assistant** | *End-to-End Web Platform* <align right>2026</align>
* Built an end-to-end AI-powered document summarization platform to process PDF files and raw text, automatically generating concise summaries for study and revision.
* Integrated Hugging Face Inference Router API using distilbart-cnn-12-6 with Abstractive Summarization; implemented hash-based duplicate detection (SHA-256) on PostgreSQL/Supabase to prevent redundant database entries; built file extraction and input tokenization pipelines.
* Achieved seamless multi-format document processing with automated text extraction, robust error handling for API/network edge cases, and reduced processing latency for large files.
* ***Technologies:*** *TypeScript, Next.js (App Router), React, Tailwind CSS, Supabase (PostgreSQL), Hugging Face Transformers API (distilbart-cnn-12-6), Node.js*
* ***GitHub:*** [Project Link](https://github.com/NguyenHoangBao0211/reviselyt)

**Reviselyt: Hệ thống Tóm tắt Tài liệu & Hỗ trợ Học tập bằng AI** | *Nền tảng Web End-to-End* <align right>2026</align>
* Xây dựng nền tảng tóm tắt tài liệu tự động bằng AI từ đầu đến cuối (End-to-End), xử lý các định dạng file PDF và văn bản thô nhằm tự động trích xuất kiến thức cô đọng phục vụ việc ôn tập.
* Tích hợp Hugging Face Inference Router API với mô hình distilbart-cnn-12-6 áp dụng kỹ thuật Tóm tắt Diễn dịch (Abstractive Summarization); thiết lập cơ chế kiểm tra và chặn trùng lặp file dựa trên mã băm (Hash) lưu tại Supabase/PostgreSQL; xây dựng pipeline trích xuất văn bản và tiền xử lý token.
* Tối ưu hóa khả năng xử lý đa định dạng tài liệu với tỷ lệ trích xuất chính xác cao, xử lý ngoại lệ mượt mà cho các lỗi kết nối/API, giúp giảm thời gian xử lý và tóm tắt file dung lượng lớn.
* ***Công nghệ sử dụng:*** *TypeScript, Next.js (App Router), React, Tailwind CSS, Supabase (PostgreSQL), Hugging Face Transformers API (distilbart-cnn-12-6), Node.js*
