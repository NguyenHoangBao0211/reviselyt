# Reviselyt - AI Document Summarizer & Study Assistant

Reviselyt là nền tảng tóm tắt tài liệu tự động bằng AI giúp tối ưu hóa quá trình học tập và ôn tập.

## 🚀 Tính năng chính
- **Tóm tắt PDF & Văn bản:** Tự động trích xuất ý chính bằng AI (distilbart-cnn-12-6).
- **Chống trùng lặp:** Dùng SHA-256 Hash kiểm tra tài liệu trên Supabase/PostgreSQL.
- **Quản lý tài liệu:** Lưu trữ và xem lại lịch sử các bản tóm tắt trên Dashboard.

## 🛠️ Công nghệ sử dụng
- **Frontend/Backend:** Next.js (App Router), TypeScript, Tailwind CSS
- **Database:** Supabase (PostgreSQL)
- **AI Integration:** Hugging Face Inference Router API
